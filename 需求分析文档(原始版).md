# 人工需求分析

**核心功能MVP：**
1)用户可以自由输入一个他想学的知识（可以是一句话、一段话、甚至是一个文档/一个视频/一个网页）
2)AI自动从全网或者特定的信息源获取知识（比如网络搜索，或者解析文档/视频/网页）
3)AI自动生成交互式问答闯关的题目、选型和答案，通过提问引导来增加用户学习的趣味性
4)用户可以自由答题闯关，每道题目选择后立刻给出正确答案和知识讲解（答错也有知识讲解）
5)通过之后生成一个知识总结和复盘报告，用户可以分享给好友
6)用户可以在后台分析自己所学的知识和答题情况，便于日后复盘，还可以生成报告
**扩展功能：**
1)允许输入知识库来生成题目(RAG)：可以用于企业培训、特定题库准备、考试复习等等
2)AI生图，比如应用到题目和选型中：可以根据知识的类型选择是否生成图片，可以用于刷英语单词等
3)充值VIP进行盈利
4)多人对战PK、排行榜、数字人、语音读题

# 需求分析文档1

### 1. 产品核心价值与亮点思路验证

**可靠性验证**：本项目通过将“获取”与“测验”合二为一，解决了学习中的“幻觉学习”（以为读了就是会了）痛点。调研显示，游戏化学习和AI测验市场均处于爆发期，且现有竞品在全网动态信息摄取方面存在空白。

**亮点思路扩展**：

- **“影子对手”异步对战**：为了解决匹配人数不足的问题，引入“影子对战”模式，让用户与好友的历史最佳答题记录进行对战，实现跨时空的竞争感 。
- **知识迁移挑战**：AI不仅从当前输入出题，还会引导用户将当前知识与之前学过的知识进行关联对比，生成“跨领域题目”，锻炼用户的深度思考能力。
- **浏览器插件一键转化**：开发浏览器扩展，用户在浏览技术博客或学术论文时，点击插件即可直接开启侧边栏闯关，极大缩短了学习链路。

### 2. 详细功能需求 (Functional Requirements)

#### 3.1 用户端核心流程

- **输入模块**：支持文本输入框、文件上传（PDF/DOCX）、网页链接解析、YouTube链接。
- **解析反馈**：AI 解析期间显示实时进度及内容大纲，允许用户筛选关心的知识子集。
- **闯关界面**：卡片式答题交互，支持即时判断。正确则前进，错误则弹出知识点解析卡。
- **复盘中心**：展示当前知识点的掌握曲线。提供“错题重练”入口。
- **社交分享**：支持生成长图海报，展示知识勋章与挑战等级。

#### 3.2 扩展功能模块

- **RAG 知识库管理**：用户可创建多个私有文件夹（知识库），AI 生成的题目将仅从该范围内检索并确保 100% 事实对齐。
- **多模态增强**：
  - **生图策略**：根据学科类型（如生物、外语）自动配置生图频率。
  - **语音交互**：支持 TTS（文字转语音）读题及语音识别答题。
- **对战系统**：
  - **排行榜**：基于知识领域、地域、学校进行多维度排名 。
  - **1v1 房间**：支持通过 Socket.io 实现的实时在线对战 。

#### 3.3 管理后台与分析

- **用户画像分析**：统计用户的学习习惯、活跃时段、知识薄弱项。
- **API 消耗监控**：监控 Token 消耗及资源成本，用于动态调整定价策略。

## 项目实施建议与未来演进

本项目不仅是一个工具，更是一个“认知伴侣”。在开发实现过程中，应优先保证“输入解析”的鲁棒性和“题目生成”的逻辑性。初步验证想法后，应迅速通过扩展功能中的“RAG模式”切入企业或垂直教育市场。

未来的演进方向可考虑与 VR/AR 设备结合，打造沉浸式的 3D 知识闯关环境，或开发基于手机摄像头的“扫一扫出题”功能。通过在认知科学、大语言模型和游戏化设计三个领域的深度交叉，本项目完全具备在竞争激烈的 EdTech 市场开辟蓝海的实力。

# 需求分析文档2

#### 1. 产品定位

一款基于 AI 的个性化互动学习工具，将任何长文本、网页、视频或文档一键转化为游戏化的闯关测验，帮助用户以最低的心理门槛完成碎片化学习与知识内化。

#### 2. 目标用户画像

- **泛知识焦虑者 / 终身学习者：** 收藏了大量干货文章/视频但从未点开看的人。
- **备考学生 / 考证党：** 需要将教材或特定题库转化为刷题模式的人。
- **企业内训 / HR（扩展期）：** 需要快速将公司操作手册转化为员工培训测验的人。

#### 3. 核心功能规划 (MVP 阶段)

- **输入模块：** 支持纯文本粘贴、URL（微信公众号/知乎/通用网页）、基础文档（PDF/Word）上传。
- **AI 处理引擎：** * 知识点抽取。
  - 题型生成（单选、多选、判断）。
  - 干扰项生成（AI需生成具有迷惑性的错误选项以保证挑战性）。
- **闯关交互模块：** 进度条显示、即时正误反馈、趣味性/人设化的知识点解析。
- **结算与分享模块：** 知识雷达图、复盘总结、生成适合朋友圈/小红书分享的精美打卡海报。
- **个人中心：** 历史学习记录、错题本、基础数据统计。

#### 4. 商业模式与扩展 (VIP 盈利设计)

- **免费版 (Freemium)：** 每日限制转换次数（如3次），限制输入文本长度（如5000字内），基础 AI 讲解。
- **VIP 订阅：** 解锁超长文档/音视频解析，解锁生图/语音读题功能，解锁特定 AI 导师人设，导出错题本（PDF/Anki格式）。

#### 5. 核心风险提示

- **版权问题：** 解析他人付费内容（如付费专栏）可能存在一定的合规风险，需在用户协议中规避。
- **API 成本：** 大量长文本输入会导致大模型 API 调用费用激增，需要严格计算免费额度与 VIP 定价的利润差。

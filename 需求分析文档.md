# AI 互动闯关学习助手：需求分析文档

## 1. 需求背景

在数字化信息爆炸的时代，用户面临“收藏从未阅读”和“读过即忘”的**幻觉学习**痛点。

- **产品定位**：一款基于 AI 的个性化互动学习工具，将碎片化的长文本、网页、视频或文档一键转化为游戏化的闯关测验。
- **核心价值**：通过“以考代练”打破认知幻觉，将被动输入转变为主动输出，降低碎片化学习的心理门槛，提升知识内化效率。
- **目标用户**：
  - **终身学习者**：解决干货堆积、阅读效率低的问题。
  - **备考学生**：针对特定教材或笔记快速生成刷题库。
  - **企业 HR/内训师**：快速将手册转化为员工考核题目。

------

## 2. MVP 最小可行功能点 (核心优先)

为了保证能够“一口气完成”并快速上线验证，MVP 阶段仅保留支撑闭环的最核心功能：

- **全能输入终端**：
  - 文本直接粘贴（支持 5000 字以内）。
  - 通用网页 URL 解析（如公众号、知乎、技术博客）。
  - 基础文档上传（PDF/Word 格式）。
- **AI 智能出题引擎**：
  - **核心逻辑**：自动提取知识点，生成单选题、判断题。
  - **迷惑项生成**：AI 需生成具有逻辑相关性的干扰项，确保题目挑战性。
- **卡片式闯关交互**：
  - 即时反馈：选择后立刻显示正误，并弹出“人设化”的知识解析卡片。
  - 进度可视：顶部显示关卡进度条。
- **复盘与结算**：
  - 生成知识总结报告。
  - 基础错题本记录，支持再次练习。
- **基础账户系统**：支持微信/手机登录，保存个人学习历史。

## 3. 后续扩展功能点 (优先级排序)

| **优先级** | **功能模块**            | **功能描述**                                   | **核心理由**                     |
| ---------- | ----------------------- | ---------------------------------------------- | -------------------------------- |
| **P0**     | **浏览器插件**          | 一键将当前浏览的网页开启侧边栏闯关。           | 极大缩短学习链路，提高留存。     |
| **P0**     | **RAG 个人知识库**      | 用户上传多个文档组成私有库，题目仅从库内生成。 | 确保事实对齐，适用于企业和考试。 |
| **P1**     | **视频/多媒体解析**     | 解析 YouTube、B站视频或播客录音。              | 覆盖更广的学习场景。             |
| **P1**     | **社交对战 (影子模式)** | 与好友的历史记录进行异步 1v1 对战。            | 增加趣味性和裂变能力。           |
| **P2**     | **多模态增强**          | AI 自动生成题目插图、语音读题 (TTS)。          | 辅助外语、生物等学科理解。       |
| **P2**     | **导出功能**            | 支持将错题或题库导出为 PDF 或 Anki 格式。      | 满足用户深度的复习需求。         |

## 4. 用户操作流程图

以下是用户从进入产品到完成学习的核心路径：

```
graph TD
    A[开始学习] --> B{选择输入方式}
    B -- 粘贴文本 --> C[AI 解析内容]
    B -- 网页链接 --> C
    B -- 上传文档 --> C
    
    C --> D[生成关卡地图/进度条]
    D --> E[显示题目卡片]
    
    E --> F{用户答题}
    F -- 回答正确 --> G[显示正确反馈+知识点强调]
    F -- 回答错误 --> H[显示解析卡片+深度纠错]
    
    G --> I{是否最后一题?}
    H --> I
    
    I -- 否 --> E
    I -- 是 --> J[生成复盘报告/勋章]
    J --> K[错题存入错题本]
    J --> L[分享海报/挑战好友]
    L --> M[结束/开始下一轮]
```
